Задание:
Создайте программу на Python, которая работает с меню кафе. Программа должна хранить словарь с названиями блюд и их ценами, а также уметь выполнять базовые операции.

Условия:
Создайте словарь menu, где ключами будут названия блюд (строки), а значениями — их цены (числа).
Напишите функции для следующих действий:
add_dish(name, price): добавляет блюдо и его цену в меню.
remove_dish(name): удаляет блюдо из меню.
get_price(name): возвращает цену конкретного блюда.
display_menu(): выводит весь список блюд с ценами.
Пример работы программы:
python
Копировать код
add_dish("Салат Цезарь", 250)
add_dish("Суп Борщ", 150)
display_menu()

# Вывод:
# Салат Цезарь: 250 руб
# Суп Борщ: 150 руб
