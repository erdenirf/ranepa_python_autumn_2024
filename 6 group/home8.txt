Задание: Учет погоды в разных городах

Напишите программу для учета погоды в разных городах. Программа должна позволять добавлять информацию о текущей погоде, обновлять данные и выводить прогнозы по городам.

Требования:
Создайте словарь, где ключами будут названия городов, а значениями — кортежи с информацией о погоде:

Температура (float),
Влажность (int),
Ветер (float).
Реализуйте следующие функции:

add_city_weather(city_name, temperature, humidity, wind_speed): добавляет данные о погоде для города. Параметры:

city_name (str): название города.
temperature (float): температура в градусах Цельсия.
humidity (int): влажность в процентах.
wind_speed (float): скорость ветра в м/с.
update_city_weather(city_name, temperature=None, humidity=None, wind_speed=None): обновляет данные о погоде для города. Параметры:

city_name (str): название города.
temperature (float, optional): новая температура.
humidity (int, optional): новая влажность.
wind_speed (float, optional): новая скорость ветра.
get_weather(city_name): выводит информацию о погоде в указанном городе.

get_all_weather(): выводит данные о погоде во всех городах.

Используйте словарь для хранения данных о погоде.

Создайте меню, которое позволяет добавлять данные о погоде, обновлять информацию, просматривать текущие данные по городам.

Пример работы программы:
python
Копировать код
# Пример словаря с погодой в городах
weather_data = {

    'Санкт-Петербург': (8.0, 80, 4.1),
    'Новосибирск': (-5.0, 90, 6.0)
}

# 1. Добавление данных о погоде: add_city_weather("Екатеринбург", 2.5, 70, 2.5)

# 3. Получение данных о погоде для города: get_weather("Санкт-Петербург")
# 4. Вывод всех данных о погоде: get_all_weather()
