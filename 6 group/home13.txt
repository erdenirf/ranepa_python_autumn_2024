Задание: Управление списком задач

Создайте программу для управления списком задач (TODO list). Программа должна позволять добавлять, удалять и изменять задачи, а также помечать задачи как выполненные.

Требования:
Создайте список задач, где каждая задача будет представлена в виде кортежа, содержащего:

Название задачи (str),
Описание задачи (str),
Статус задачи (True/False, где True — задача выполнена, False — задача не выполнена).
Реализуйте следующие функции:

add_task(title, description): добавляет новую задачу в список. Параметры:

title (str): название задачи.
description (str): описание задачи.
delete_task(title): удаляет задачу из списка по названию. Параметр:

title (str): название задачи.
mark_as_done(title): помечает задачу как выполненную. Параметр:

title (str): название задачи.
mark_as_undone(title): помечает задачу как невыполненную. Параметр:

title (str): название задачи.
get_task_info(title): выводит информацию о задаче. Параметр:

title (str): название задачи.
get_all_tasks(): выводит все задачи, включая их статус.

Задачи должны быть добавлены в список с начальным статусом "не выполнена".

Программа должна поддерживать работу с несколькими задачами и быть способной отслеживать их состояние.

Пример работы программы:
