Задание: Учет задач для сотрудников

Создайте программу для учёта задач, выполняемых сотрудниками. Программа должна позволять добавлять задачи, назначать задачи сотрудникам, отслеживать статус задач, обновлять статус и получать отчёт о выполнении всех задач.

Требования:
Создайте словарь, где ключами будут имена сотрудников, а значениями — списки кортежей с информацией о задачах. Каждый кортеж должен содержать:

Описание задачи (str),
Дата завершения (str в формате ДД.ММ.ГГГГ),
Статус задачи (str, например, "в процессе", "выполнено").
Реализуйте следующие функции:

add_employee(employee_name): добавляет нового сотрудника в систему. Параметры:

employee_name (str): имя сотрудника.
add_task(employee_name, task_description, due_date): добавляет задачу сотруднику. Параметры:

employee_name (str): имя сотрудника.
task_description (str): описание задачи.
due_date (str): дата завершения задачи.
update_task_status(employee_name, task_description, new_status): обновляет статус задачи для конкретного сотрудника. Параметры:

employee_name (str): имя сотрудника.
task_description (str): описание задачи.
new_status (str): новый статус задачи.
get_tasks(employee_name): выводит все задачи для конкретного сотрудника.

get_report(): выводит отчёт обо всех сотрудниках и их задачах, включая их текущий статус.

Используйте словарь для хранения данных о сотрудниках и их задачах, где ключи — это имена сотрудников, а значения — списки с кортежами задач.

Создайте меню, которое позволяет пользователю добавлять сотрудников, добавлять задачи, обновлять статус задач и получать отчёт обо всех задачах.

# 3. Обновление статуса задачи: update_task_status("Майя", "Составить план проекта", "в процессе")
# 4. Получение задач для сотрудника: get_tasks("Иван")
# 5. Получение отчёта обо всех задачах: get_report()
