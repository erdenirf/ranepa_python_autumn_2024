Задание 1: Фильтрация словаря по значению
Напишите программу, которая принимает словарь с целочисленными значениями и фильтрует его так, чтобы в итоговом словаре остались только те элементы, где значение больше заданного порога.

Пример:

original_dict = {'a': 10, 'b': 5, 'c': 20, 'd': 2}
threshold = 6
Ожидаемый результат: {'a': 10, 'c': 20}.

Задание 2: Множество с проверкой наличия элементов
Напишите программу, которая принимает два множества и проверяет, является ли одно множество подмножеством другого. Программа должна вернуть булево значение (True или False).

Пример:

set1 = {1, 2, 3}
set2 = {1, 2, 3, 4, 5}
Ожидаемый результат: True (set1 является подмножеством set2).
