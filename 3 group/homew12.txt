Задание 1: Извлечение уникальных значений из списка словарей
Напишите программу, которая принимает список словарей, где каждый словарь имеет одинаковые ключи, но разные значения. Программа должна создать множество, которое будет содержать только уникальные значения для каждого ключа.

Пример:

list_of_dicts = [{'a': 1, 'b': 2}, {'a': 3, 'b': 2}, {'a': 1, 'b': 4}]
Ожидаемый результат:

{'a': {1, 3}, 'b': {2, 4}}
Задание 2: Сумма чисел в множестве, кроме минимального и максимального
Напишите программу, которая принимает множество чисел. Программа должна вычислить и вернуть сумму всех элементов множества, за исключением минимального и максимального значений.

Пример:

numbers = {5, 1, 8, 3, 7}
Ожидаемый результат: 18 (сумма чисел 5 + 3 + 7).
