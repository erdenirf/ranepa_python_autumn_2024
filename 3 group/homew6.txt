Задание 1: Анализ словаря
Напишите программу, которая запрашивает у пользователя словарь с данными о студентах в формате:
Имя:Возраст, например:

Алиса:21, Борис:19, Виктория:22, Дмитрий:20
Программа должна:

Вывести средний возраст студентов.
Найти самого старшего и самого младшего студента.
Пример ввода/вывода:

Введите данные о студентах (Имя:Возраст): Алиса:21, Борис:19, Виктория:22, Дмитрий:20
Средний возраст: 20.5
Самый старший студент: Виктория (22)
Самый младший студент: Борис (19)

Задание 2: Работа с множествами
Напишите программу, которая:

Запрашивает у пользователя два набора чисел, вводимых через пробел.
Определяет:
Какие числа есть в обоих наборах.
Какие числа уникальны для каждого набора.
Все числа, которые входят хотя бы в один из наборов.
Пример ввода/вывода:

Введите первый набор чисел: 1 2 3 4 5  
Введите второй набор чисел: 4 5 6 7 8  

Общие числа: {4, 5}  
Числа уникальные для первого набора: {1, 2, 3}  
Числа уникальные для второго набора: {6, 7, 8}  
Все числа: {1, 2, 3, 4, 5, 6, 7, 8}
