Задание 1: Работа со словарями
Создайте словарь с такими данными:

students = {
    "Алексей": 85,
    "Мария": 90,
    "Иван": 78,
    "Екатерина": 92
}
Действия для выполнения:

Добавьте студента "Дмитрий" с оценкой 88.
Обновите оценку для студента "Иван" до 80.
Удалите студента "Екатерина".
Найдите студента с максимальной оценкой (ожидается, что это "Мария").
Подсчитайте среднюю оценку (учитывая изменения).


Задание 2: Работа с множествами
Даны множества:

A = {"Алексей", "Мария", "Иван", "Екатерина"}
B = {"Мария", "Екатерина", "Дмитрий", "Сергей"}
Действия для выполнения:

Найдите пересечение множеств A и B (кто посетил обе лекции).
Найдите разницу множеств A - B (кто посетил только лекцию по математике).
Проверьте, является ли A подмножеством B.
Подсчитайте общее количество уникальных студентов, посетивших хотя бы одну лекцию.
