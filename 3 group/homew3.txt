Задание 1: Работа со словарями
Создайте словарь с такими данными:

students = {
    "Сергей": 75,
    "Ольга": 88,
    "Анна": 95,
    "Владимир": 82
}
Действия для выполнения:

Добавьте студента "Елена" с оценкой 91.
Обновите оценку для студента "Сергей" до 78.
Удалите студента "Анна".
Найдите студента с максимальной оценкой (ожидается, что это "Елена").
Подсчитайте среднюю оценку (учитывая изменения).

Задание 2: Работа с множествами
Даны множества:

A = {"Сергей", "Ольга", "Анна", "Владимир"}
B = {"Анна", "Елена", "Дмитрий", "Сергей"}
Действия для выполнения:

Найдите пересечение множеств A и B (кто посетил обе лекции).
Найдите разницу множеств B - A (кто посетил только лекцию по физике).
Проверьте, является ли B подмножеством A.
Подсчитайте общее количество уникальных студентов, посетивших хотя бы одну лекцию.
