Задание 1: Словарь с длинами строк
Напишите программу, которая принимает список строк и создает словарь, в котором ключами являются строки, а значениями — длины этих строк. Программа должна вернуть этот словарь.

Пример:

strings = ["apple", "banana", "cherry"]
Ожидаемый результат: {'apple': 5, 'banana': 6, 'cherry': 6}.

Задание 2: Множество и подсчет вхождений
Напишите программу, которая принимает список чисел и возвращает множество, содержащее только те числа, которые встречаются в списке больше одного раза. Используйте множество для поиска этих чисел.

Пример:

numbers = [1, 2, 3, 4, 5, 2, 3, 6]
Ожидаемый результат: {2, 3}.
